# Pulse Chat

Мессенджер в стиле Telegram: личные и групповые чаты, сообщения в реальном времени, статус «в сети», индикатор набора текста.

**Стек:** Node.js (без npm-зависимостей) + HTML/CSS/JS.

## Запуск

**Двойной клик** по `start.bat` — скрипт сам найдёт Node.js (в том числе встроенный в Cursor). Если сервер уже запущен, откроется браузер.

**Остановить сервер:** `stop.bat` (освобождает порт 3847).

Если `start.bat` пишет, что `node` не найден:

1. Установите [Node.js LTS](https://nodejs.org/) и перезапустите `start.bat`, **или**
2. Запустите PowerShell-скрипт: правый клик → «Выполнить с помощью PowerShell» на `start.ps1`, **или**
3. В терминале Cursor: `node server.js` из папки проекта.

```bash
node server.js
```

Откройте в браузере: **http://localhost:3847**

## Возможности

- Регистрация и вход по имени пользователя
- Поиск пользователей и личные чаты
- Личные чаты
- Ответ, пересылка, реакции, удаление (ПКМ по сообщению)
- Сообщения в реальном времени (Server-Sent Events)
- Список чатов с превью последнего сообщения
- Счётчик непрочитанных
- Статус онлайн / последний визит
- Индикатор «печатает…»
- Уведомления: звук, тосты в приложении, системные (Windows / браузер)
- Футуристичный glass UI
- Голосовые сообщения (запись с микрофона)
- Смена отображаемого имени и username в профиле
- Аудио- и видеозвонки (WebRTC, личные чаты, до 8 участников)
- Во время звонка: кнопка ➕ — пригласить другого пользователя

### Звонки на одном компьютере

Windows часто отдаёт **один микрофон только одной вкладке**. Для теста в двух браузерах:

1. В **первом** окне — звоните как обычно (с микрофоном).
2. Во **втором** — включите «**Только слушать**» перед принятием звонка.
3. Используйте **наушники**, чтобы не было эха с колонок.

## Данные

Всё хранится в папке `data/` (JSON). Для сброса удалите `data/store.json`.

## Несколько пользователей

Запустите сервер один раз. Откройте сайт в разных браузерах или вкладках инкогнито — зарегистрируйте разных пользователей и переписывайтесь.
